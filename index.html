Instructions & User Guide
How to use this system — read before starting

🌳 Welcome to Southern Oak Realty LLC Property Manager

This system tracks your entire real estate investment lifecycle — from purchase through disposition — and generates tax documents for your CPA. All data is stored in your browser AND synced to Google Sheets for multi-user access.

📋 Step-by-Step Setup
  1. Connect Google Sheets — Click "Google Sheets Setup" in the top bar. Paste your Sheet ID. This enables 5-user shared access.
  2. Add Properties — Go to Purchase Log. Enter each property's address, purchase price, closing costs, land value, and building value.
  3. Log Remodels/CapEx — Under Remodel/CapEx, record all capital improvements (adds to basis, depreciated separately).
  4. Record Maintenance — Log all repairs under Maintenance (fully deductible in the year paid).
  5. Enter Rental Income — Each month, enter rent received per property under Income & Rent.
  6. Log Operating Expenses — Insurance, HOA, property tax, utilities, management fees go under Operating Expenses.
  7. Review Depreciation — The system auto-calculates 27.5-year straight-line depreciation for residential, 39-year for commercial.
  8. Generate CPA Report — At year-end, go to CPA Report, select the tax year, and print/save PDF for your CPA.
💡 Key Tax Rules (2026–2050)
  • Depreciation: Residential rental = 27.5 years. Commercial = 39 years. Land is NOT depreciated.
  • Repairs vs. CapEx: Routine repairs are expenses; improvements adding value or extending life are capitalized and depreciated.
  • Passive Activity Loss Rules: Rental losses may be limited. Active participants can deduct up to $25,000 (phases out $100K–$150K AGI).
  • Schedule E: All rental income/expenses flow to Schedule E (personal return) or Form 1065 (partnership/LLC).
  • QBI Deduction: LLC may qualify for 20% qualified business income deduction.
  • Self-Employment Tax: Typically not owed on rental income unless real estate professional.
  • 1031 Exchange: Log deferred gains when a property is sold and proceeds reinvested.
  • Cost Segregation: Flag items eligible for accelerated depreciation (5, 7, or 15-year property).
  • State Taxes: Alabama has a flat 5% income tax rate on rental income. File Form 20S for LLC.
👥 Multi-User Access (Google Sheets)
  1. Open Google Sheets and create a new spreadsheet named "Southern Oak Realty - Data".
  2. Go to Share → add up to 5 email addresses with Editor access.
  3. Copy the Sheet ID from the URL (the long string between /d/ and /edit).
  4. Paste the Sheet ID into the Google Sheets Setup modal in this app.
  5. Each user bookmarks this HTML file or hosts it on Google Drive as a shared file.
  6. Data entered in this app syncs to the sheet; any user can pull latest data using the Sync button.
🖨 Generating Your CPA Tax Package
  1. At year-end (or when requested), navigate to CPA Report in the sidebar.
  2. Select the tax year from the year selector.
  3. Review all figures — income, deductions, depreciation, net income.
  4. Click "Print CPA Report" in the top bar, or use Ctrl+P / Cmd+P.
  5. Save as PDF and email to your CPA, or print two copies (one for CPA, one for your records).
  6. The report includes all Schedule E data, depreciation schedules, and supporting detail.

🔗 Connect Your Google Sheet

How to Get Your Sheet ID

  1. Go to sheets.google.com and create a new spreadsheet.
  2. Name it: Southern Oak Realty - Data
  3. Click Share → Add each of the 5 users' Gmail addresses as Editors.
  4. Copy the Sheet ID from the URL:
    https://docs.google.com/spreadsheets/d/[SHEET ID IS HERE]/edit
  5. Paste the Sheet ID above and click Save.
  6. Each user should bookmark this HTML file or upload it to a shared Google Drive folder.

Note: To enable the API, go to Google Cloud Console → APIs → Enable Google Sheets API → Create OAuth credentials and replace the placeholder API key in this file's script section.

📱 Access From Any Device: Host this HTML file on Google Drive or a shared hosting service. All users open the same URL and data syncs through the connected Google Sheet.
